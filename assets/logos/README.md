# 🇺 Logo Assets

## Available Formats

### Primary Logo
- `aiotize-primary-light.svg` - Primary logo for light backgrounds
- `aiotize-primary-dark.svg` - Primary logo for dark backgrounds
- `aiotize-primary.png` - PNG version (multiple sizes: 512px, 1024px, 2048px)

### Wordmark
- `aiotize-wordmark-light.svg`
- `aiotize-wordmark-dark.svg`

### Icon/Symbol
- `aiotize-icon.svg` - Square icon/symbol only
- `aiotize-icon-rounded.svg` - Icon with rounded corners

## Usage Guidelines

### Minimum Size
- Digital: 48px height minimum
- Print: 0.5 inch height minimum

### Clear Space
Maintain a minimum clear space around the logo equal to the height of the "A" in Aiotize.

### Background Colors
**Approved backgrounds:**
- White (#FFFFFF)
- Light gray (#F8FAFC)
- Dark blue (#0A1628)
- Brand blue (#0066FF)

### Don'ts
❌ Don't rotate or distort the logo  
❌ Don't add effects (shadows, glows, gradients)  
❌ Don't change logo colors  
❌ Don't place on busy backgrounds  
❌ Don't use outdated versions
